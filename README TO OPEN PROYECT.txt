La aplicación Álbumes se realizó utilizando el framework "Laminas mvc". 

El directorio en el cual se almacenó el codigo de la aplicación se localiza en "module/Cporders/".

El script de la base de datos, así como el esquema e inserts se encuentran en la carpeta "data/" realizados con SQLite.

Para poder ver lo realizado, una vez que extraemos el proyecto en la carpeta de nuestro host, entraremos a las siguientes URL's:

http://localhost/laminas-mvc-tutorial/public/cporders/supplier
http://localhost/laminas-mvc-tutorial/public/cporders/customer
http://localhost/laminas-mvc-tutorial/public/cporders/product
http://localhost/laminas-mvc-tutorial/public/cporders/order
